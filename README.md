# 思考と反復のギフト（ランキング）

固定200秒で完結するランキング形式ニコ生ゲームです。
成長フェーズでは「思考＝神経衰弱」「反復＝落ちもの合体」をタブで切り替えつつ同時進行し、獲得したスキルで自動戦闘に挑みます。

## 特徴
- 200秒固定（タイトル12 / 成長120 / 戦闘60 / 結果8）
- 成長フェーズ：神経衰弱でスキル獲得（高位は条件未達だとスカスキル）
- 成長フェーズ：落ちもの合体でスキルLv強化、スカ同士は対消滅
- 相互干渉：落下操作でカード再シャッフル＆条件進行、欠片でMP回復＆昇格
- 戦闘フェーズ：完全自動 + 倍速(1〜5) + 逃走でスコア確定（HP0で0点）
- 「才能の壁」イベント：条件を満たすと大きくスコアが伸びる

## 操作
- タブ切替：画面上部の「思考/反復」
- 思考（神経衰弱）：カードをタップ
- 反復（落ちもの）：
  - 落下物をドラッグで左右移動
  - 画面タップで落下開始（落下操作でカードが再シャッフル）
- 戦闘：
  - 倍速ボタン(1x〜5x)
  - 逃走（スコア確定）

## ルール（スコア）
- 撃破数・敵Lv・連勝倍率・HP残量でスコア算出
- HP0になるとスコア0

## 実行環境
- Akashic Engine（ニコ生ゲーム/ランキング）
